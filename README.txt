深度学习导论课上的matlab实现卷积神经网络的代码，手写数字识别作业

W_update_xxxxx:权值更新函数
showACC_xxxxx:根据储存的结果快速给出正确率
results_xxxxx:训练结果储存，保存有各类网络训练出的权值

OneHiddenlayer  单层隐藏层的简单神经网络
NoHiddenlayer   无隐藏层的简单神经网络
CNNTwoHiddenLayers 判别网络有两层隐层的CNN
CNNdropout  使用dropout方法的CNN
CNN 最终得出训练结果最好的CNN网络
